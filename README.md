# Стрельников Максим (ИСТ04/25б). Курсовая работа по теме "9. Модель социальной сети"
## а) Титульный лист
**Тема:** Модель социальной сети

**Дисциплина:** Языки и методы программирования

**Выполнил:** Стрельников Максим Эдуардович

## б) Постановка задачи
Разработка программной модели социальной сети на языке программирования Python с применением принципов объектно-ориентированного программирования (ООП) для демонстрации навыков проектирования классовых структур, работы с файловыми системами и создания интерактивного пользовательского интерфейса.

## Требования к проекту
1. **Управление пользователями**
- Регистрация новых пользователей с уникальным логином
- Авторизация по логину и паролю
- Выход из аккаунта
- Смена пароля
2. **Социальные функции**
- Добавление пользователей в друзья
- Удаление из друзей
- Просмотр списка друзей с именами и ID
3. **Публикации**
- Создание постов на своей стене
- Просмотр своих публикаций
- Просмотр ленты новостей (посты друзей)
4. **Сообщения**
- Отправка личных сообщений другим пользователям
- Просмотр входящих сообщений
- Просмотр исходящих сообщений
- Автоматическая отметка о прочтении входящих сообщений
5. **Работа с файлами**
- Автоматическое сохранение базы данных в network_db.json
- Экспорт профиля пользователя в текстовый файл username_profile.txt
